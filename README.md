# mSWEEP2
[mSWEEP](https://github.com/PROBIC/mSWEEP) but better.

Under development -- if you want to contribute or have suggestions, get in touch.

See the [status tracker](https://github.com/tmaklin/mSWEEP2/issues/1) for what's
planned currently.

# Installation
## Building from source
### Requirements
- Rust compiler
- C++17 compliant compiler.
- cmake (v3.11 or newer)
- git

### Compiling
Clone the mSWEEP repository
```
git clone https://github.com/tmaklin/mSWEEP2.git
```
enter the directory and run
```
> mkdir build
> cd build
> cmake ..
> make
```
